● توضیحات

← این اسکریپت پکیچ PassWall2 را به راحتی روی فریمور OpenWrt نصب می‌کند.

← با تفکیک ترافیک، سایت‌های داخلی ایران بدون نیاز به قطع کردن پروکسی باز می‌شوند.

← در صفحه تنظیمات PassWall2، وضعیت لحظه‌ای اینترنت و عبور از فیلترینگ قابل مشاهده است.

← اسکریپت همیشه آخرین نسخه رسمی را نصب می‌کند.

● ویژگی‌ها

  • نصب از منابع رسمی
  • تفکیک ترافیک ایران و جهانی
  • پایگاه داده کامل GeoIP / Geosite
  • تنظیم DNS و منطقه زمانی تهران
  • ایجاد شانت و قواعد هوشمند
  • بنر حرفه‌ای PassWall2

● پیش‌نیازها

  • روتر OpenWrt
  • اینترنت فعال
  • SSH با سطح root

● نصب سریع (یک فرمان)

rm -f Passwall-IR.sh && wget https://raw.githubusercontent.com/saeed9400/IRAN_Passwall2/main/Passwall-IR.sh && chmod +x Passwall-IR.sh && sh Passwall-IR.sh

● نصب دستی

1. اتصال به SSH:

ssh root@192.168.1.1

2. دانلود اسکریپت:

wget https://raw.githubusercontent.com/saeed9400/IRAN_Passwall2/main/Passwall-IR.sh -O Passwall-IR.sh

3. اجرا:

chmod +x Passwall-IR.sh && ./Passwall-IR.sh
