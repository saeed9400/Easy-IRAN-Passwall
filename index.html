● توضیحات

← این اسکریپت پکیج PassWall2 را به راحتی روی فریمور OpenWrt نصب می‌کند.

← با تفکیک ترافیک، میتوانید سایت‌های داخلی ایران را بدون نیاز به قطع کردن پروکسی استفاده کنید.

← در صفحه تنظیمات PassWall2 می‌توانید وضعیت لحظه‌ای تفکیک ترافیک، اینترنت و عبور از فیلترینگ را ببینید.

← اسکریپت همیشه آخرین نسخه رسمی PassWall2 را نصب می‌کند.

● ویژگی‌ها

  • نصب PassWall2 فقط از منابع رسمی
  • نصب پکیج‌های تفکیک ترافیک از سورس‌های معتبر
  • پایگاه داده کامل GeoIP / Geosite ایران
  • تنظیم منطقه زمانی تهران و DNS عمومی
  • قواعد هوشمند برای ترافیک ایران و جهانی
  • ایجاد Shunt Node برای مدیریت ترافیک
  • بنر اختصاصی کاربران ایرانی PassWall2

● پیش‌نیازها

  • روتر با OpenWrt
  • اینترنت فعال
  • دسترسی SSH با سطح root

● نصب سریع (یک فرمان)

rm -f Passwall-IR.sh && wget https://raw.githubusercontent.com/saeed9400/IRAN_Passwall2/main/Passwall-IR.sh && chmod +x Passwall-IR.sh && sh Passwall-IR.sh

● نصب دستی مرحله به مرحله

1. اتصال به SSH:

ssh root@192.168.1.1

2. دانلود اسکریپت:

wget https://raw.githubusercontent.com/saeed9400/IRAN_Passwall2/main/Passwall-IR.sh -O Passwall-IR.sh

3. اجرای اسکریپت:

chmod +x Passwall-IR.sh && ./Passwall-IR.sh

● نکات مهم

  • روتر باید اینترنت داشته باشد.
  • تهیه نسخه پشتیبان از تنظیمات قبلی توصیه می‌شود.
  • استفاده شخصی و غیرتجاری آزاد است.
