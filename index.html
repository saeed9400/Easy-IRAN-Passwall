IRAN Passwall2

نصب آسان و هوشمند PassWall2 با تفکیک ترافیک ایران

توضیحات پروژه

این اسکریپت پکیج PassWall2 را به‌راحتی و فقط از منابع رسمی روی فریمور OpenWrt نصب می‌کند.

با تفکیک هوشمند ترافیک، سایت‌های داخلی ایران بدون نیاز به قطع پروکسی قابل دسترسی هستند.

ویژگی‌های کلیدی

نصب از منابع رسمی

تفکیک ترافیک ایران

دیتابیس کامل GeoIP/Geosite

منطقه زمانی تهران + DNS

قوانین هوشمند Shunt

بنر اختصاصی ایرانی

پیش‌نیازها

  • روتر با فریمور OpenWrt
  • اتصال فعال به اینترنت
  • دسترسی SSH با کاربر root

نصب سریع (توصیه شده)

rm -f Passwall-IR.sh && wget https://raw.githubusercontent.com/saeed9400/IRAN_Passwall2/main/Passwall-IR.sh && chmod +x Passwall-IR.sh && sh Passwall-IR.sh

نکات مهم

  • روتر باید به اینترنت متصل باشد
  • پیش از نصب از تنظیمات فعلی بک‌آپ بگیرید
  • استفاده شخصی و غیرتجاری کاملاً آزاد است

Project Description

This script installs the PassWall2 package easily and only from official sources on OpenWrt firmware.

With smart traffic splitting, you can access Iranian domestic websites without disabling the proxy.

Key Features

Official Sources Only

Iran Traffic Splitting

Full GeoIP/Geosite DB

Tehran Timezone + DNS

Smart Shunt Rules

Persian Custom Banner

Requirements

  • Router running OpenWrt
  • Active internet connection
  • SSH access as root

Quick Installation (Recommended)

rm -f Passwall-IR.sh && wget https://raw.githubusercontent.com/saeed9400/IRAN_Passwall2/main/Passwall-IR.sh && chmod +x Passwall-IR.sh && sh Passwall-IR.sh
